Блокнот zso_1well_simple_cond.ipynb 
Приведен расчет размеров зон санитарной охраны водозаборной скважины 
(по методике Орадовская А.Е., Лапшин Н.Н. Санитарная охрана водозаборов подземных вод. — М.Ж Недра, 1987, стр. 93 - 101) 
Повторены графики и таблица подбора безразмерных параметров, а так же реализованы функции вещественного бинарного поиска по значению параметра T,
так же рассмотрено построение самих зон санитарной охраны на двумерном графике.
